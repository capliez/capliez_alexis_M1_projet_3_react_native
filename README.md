# Project 3 React Native With Redux and Firebase

### Objectives

Creation of an e-commerce application

### Prerequisites

What you need to use it:
  * Node.js | npm
  * CLI Expo

### Installation

Step to install:
  1. ``` git clone https://github.com/capliez/capliez_alexis_M1_projet_3_react_native.git ```
  2. ``` npm install ```
  3. ``` create file .env ```
    * API_KEY=""
    * AUTH_DOMAIN=""
    * DATABASE_URL=""
    * PROJECT_ID="-"
    * MESSAGE_SENDER_ID=""
    * APP_ID=""
    * STORAGE_BUCKET=""
    * MEASUREMENT_id=""
  3. ``` expo start ```
